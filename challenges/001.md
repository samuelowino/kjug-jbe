# Challenge 1: Temperature Converter API

## Business Requirement
Build an API that converts temperatures between Celsius and Fahrenheit.

## Endpoints

| Method        | Endpoint                       | Description                   |
|---------------|--------------------------------|-------------------------------|
| GET |         `/convert/celsius-to-fahrenheit` | Convert Celsius to Fahrenheit |
| GET |         `/convert/fahrenheit-to-celsius` | Convert Fahrenheit to Celsius |

## Business Requirements

### Requirement 1: Accept temperature value
The system must accept a temperature value as input from the user.

### Requirement 2: Validate input exists
The system must reject requests that do not provide a temperature value, with a clear error message.

### Requirement 3: Enforce physical limits
The system must reject temperatures below absolute zero (-273.15°C or -459.67°F) because they are physically impossible.

### Requirement 4: Perform accurate conversion
The system must correctly convert between Celsius and Fahrenheit using standard mathematical formulas.

### Requirement 5: Provide clear response format
The system must return the original value, original unit, converted value, and converted unit in a structured format.

### Requirement 6: Round to two decimal places
The system must round all temperature values to two decimal places for readability.

## Acceptance Tests

# Test 1: User converts boiling point of water
GET /convert/celsius-to-fahrenheit?value=100
Expected: Returns 212.0°F

# Test 2: User converts freezing point of water
GET /convert/fahrenheit-to-celsius?value=32
Expected: Returns 0.0°C

# Test 3: User forgets to provide temperature
GET /convert/celsius-to-fahrenheit
Expected: System rejects with error message about missing value

# Test 4: User enters physically impossible temperature
GET /convert/celsius-to-fahrenheit?value=-300
Expected: System rejects because below absolute zero

# Test 5: User converts zero degrees
GET /convert/celsius-to-fahrenheit?value=0
Expected: Returns 32.0°F

# Test 6: User converts negative but valid temperature
GET /convert/fahrenheit-to-celsius?value=-40
Expected: Returns -40.0°C (the point where both scales meet)
